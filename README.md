# 🧪 Alquimia Real - Simulador de Química

¡Bienvenido a **Alquimia Real**! Un simulador químico interactivo basado en la web donde puedes experimentar con los elementos de la tabla periódica, fusionarlos mediante colisiones físicas y descubrir nuevos compuestos químicos con datos curiosos reales.

Este proyecto está construido enteramente con tecnologías web nativas (**HTML5, CSS3 y JavaScript**), lo que significa que es ultra ligero y se ejecuta directamente en cualquier navegador web o dispositivo móvil sin necesidad de instalar complementos.

---

## 🚀 Características Clave

* **Los 118 Elementos Base:** Acceso completo a toda la tabla periódica desde un panel lateral intuitivo con buscador integrado.
* **Física de Colisiones:** Sistema de arrastrar y soltar (*Drag and Drop*) asistido por matemáticas de detección radial para fusionar elementos al chocarlos.
* **Efectos Visuales Activos:** Animaciones de partículas basadas en trigonometría dinámica para simular explosiones de enlaces químicos.
* **Bitácora de Descubrimientos:** Panel de registro en tiempo real que guarda tus compuestos desbloqueados con sus respectivas fórmulas e información educativa.
* **Prevención de Bloqueos:** Flexibilidad molecular avanzada (por ejemplo, asimilación de un tercer hidrógeno al agua $H_2O + H$ para formar el catión Hidronio $H_3O$ sin romper la lógica del juego).
* **Mesa Auto-Limpiable:** Botón integrado para reiniciar el espacio de laboratorio en cualquier momento.

---

## 🛠️ Tecnologías Utilizadas

El juego está unificado en un solo entorno auto-contenido:
* **HTML5:** Estructura del lienzo de laboratorio y contenedores semánticos.
* **CSS3 (Variables & Neomorfismo):** Paleta de colores optimizada para modo oscuro (`#0f172a`), bordes brillantes y diseño adaptativo.
* **JavaScript (ES6):** Motor de colisiones por hipotenusa, control de estados de arrastre y manipulación del DOM.
* **FontAwesome v6.4.0:** Iconografía vectorial para la interfaz de usuario.

---

## 🎮 Cómo Jugar (Instrucciones)

1. **Añadir Reactivos:** Haz clic sostenido sobre cualquier elemento del inventario izquierdo y arrástralo hacia el recuadro gris de la mesa de trabajo.
2. **Reordenar el Laboratorio:** Mueve con libertad los elementos que ya están en la mesa arrastrándolos con el cursor.
3. **Provocar Reacciones:** Suelta un elemento directamente encima de otro. Si comparten afinidad electrónica, ocurrirá una reacción; si no, los elementos vibrarán en rojo indicando un enlace inviable.
4. **Consultar tus Logros:** Cambia a la pestaña **"Bitácora"** en el panel lateral para repasar las propiedades de los compuestos que has descubierto.
